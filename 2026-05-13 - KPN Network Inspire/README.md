## Try it yourself

To use the demo assets in your own Power BI project:

1. **Copy the configuration files** — Copy the `.github/` and `.vscode/` folders from `Demo files/` into the root of your Power BI Desktop (PBIP) project folder.
2. **Add team standards** — Copy `team-standards.md` into a `specs and principles/` folder in your project root. Edit the rules to match your organization's modeling conventions.
3. **Open in VS Code** — Open the project folder in VS Code with the [Power BI](https://marketplace.visualstudio.com/items?itemName=PowerBITeam.pbip-vscode) extension installed.
4. **Start using agent mode** — Open GitHub Copilot chat, switch to **Agent** mode, and select the `powerbi-developer` or `powerbi-architect` participant to start designing or building your semantic model and reports.
