# Fabric Capacities - Design System Icons

Transparent background icons for capacity presentations.

## SKU Grids
- `f2-grid` through `f2048-grid` - Capacity unit grids for each SKU size

## Timeline Bars
- `bar-interactive` - Red bar for interactive operations (5-min smoothing)
- `bar-background` - Blue bar for background operations (24-hr smoothing)  
- `bar-stacked` - Combined stacked bar

## Other
- `cu-block` - Single capacity unit
- `window-30sec` - 30-second evaluation window block
